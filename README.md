# Вакансии HH.ru

## Описание проекта

Данная программа предназначена для получения информации о вакансиях с платформы hh.ru в России. \
Она позволяет сохранять данные о вакансиях в файл и предоставляет удобные инструменты для работы с ними: добавление, 
фильтрация и удаление вакансий.

## Функциональные возможности

- Получение вакансий с платформы hh.ru через API.
- Сохранение информации о вакансиях в JSON-файл.
- Фильтрация вакансий по ключевым словам в описании.
- Получение топ N вакансий по зарплате.
- Удаление вакансий из файла.
- Валидация данных вакансий, включая проверку наличия зарплаты.

## Использование
После запуска программы вы сможете взаимодействовать с ней через консоль. Вот основные команды и их описание:

1. **Ввод поискового запроса**:
   - При запуске программы вам будет предложено ввести поисковой запрос для получения вакансий с платформы hh.ru.
   - Пример ввода: `Программист Python`

2. **Получение топ N вакансий по зарплате**:
   - После ввода поискового запроса программа запросит у вас количество вакансий, которые вы хотите получить в топе по 
зарплате.
   - Пример ввода: `5`
   - Программа выведет список из 5 вакансий, отсортированных по зарплате.

3. **Фильтрация вакансий по ключевым словам**:
   - Вы также можете ввести ключевые слова для фильтрации вакансий. Программа будет искать вакансии, в которых указанные
слова присутствуют в названии или описании.
   - Пример ввода: `удалённая работа`
   - Программа выведет вакансии, соответствующие введённым ключевым словам.

4. **Удаление вакансий**:
   - В будущем вы сможете добавить функциональность для удаления вакансий из файла. На данный момент эта функция может 
быть реализована в виде заглушки.
